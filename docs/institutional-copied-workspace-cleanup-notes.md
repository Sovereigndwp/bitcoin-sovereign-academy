# Institutional Copied Workspace Cleanup Notes

## Purpose

Clean the copied Institutional workspace by separating runtime content from support files.

## Scope

This cleanup applies only to:
- apps/institutional/source-current

It does not change the live `institutional/` folder.

## Support file types to separate
- markdown notes
- README files
- QUICKSTART files
- backup files

## Rule

Do not move runtime HTML, JS, or CSS.
Only move support and backup files out of the copied Institutional tree.
