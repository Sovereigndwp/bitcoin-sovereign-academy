# BSA Interactive Demos Cleanup Notes

## Purpose

Clean the copied BSA interactive demos workspace by separating runtime demo content from support files.

## Scope

This cleanup applies only to:
- apps/bsa/source-current/interactive-demos

It does not change the live `interactive-demos/` folder.

## Support file types to separate
- markdown notes
- README files
- AGENTS files
- backup files

## Rule

Do not move runtime demo HTML, JS, or CSS.
Only move support and backup files out of the copied demo tree.
