# Post-type templates

Copy a pattern, fill the brackets, run `npm run social:validate`. One idea per
post. Useful even if the reader never clicks. Brand voice, not personal voice.

---

## 1. one-idea
3–8 lines. State a misconception or a first principle, then the more useful truth.
```
[A common belief, in one line.]

[The more useful truth, in one line.]

[1–3 short lines of contrast, example, or consequence.]
```

## 2. mistake
One mistake, why it costs, what to do instead. No shame.
```
[The mistake, named plainly.]

[Why it matters / what it costs.]

[What to do instead — concrete, doable this week.]
```

## 3. checklist
5–7 bullets max. Each is checkable.
```
[Topic in one line.]

— [item 1]
— [item 2]
— [item 3]
— [item 4]

[One closing line that makes the point.]
```

## 4. question
One sharp question with a real answer (not rhetoric). Optional one-line context.
```
[A question that reorders the reader's priorities.]

[One line of why it matters. No answer — let them sit with it or reply.]
```

## 5. thread (Nostr 5–8 notes / X numbered)
One idea broken into steps. Each note stands alone.
```
1/ [Headline. The whole point in one line.]
2/ [Step one — a concrete example or mistake.]
3/ [Step two.]
4/ [Step three.]
5/ [Closing line that lands harder than 1/. One link OR one question — not both.]
```

## 6. lesson-link
Useful without clicking; one BSA link that deepens it.
```
[A complete, useful idea — works on its own.]

[One sentence on what the link adds.]
[https://bitcoinsovereign.academy/...]
```

## 7. zap-worthy
Practical enough that a reader may zap voluntarily. Value-for-value, never begging.
```
[A genuinely useful checklist / warning / mini-lesson.]

[If this saved you confusion or a costly mistake, a zap keeps the lessons free.]
Value for value.
⚡ sovereigndwp@getalby.com
```

## 8. advisor
Professional register for RIAs, lawyers, family offices. Precise, non-hype.
```
[A distinction advisors' clients get wrong.]

[The structural explanation — 2–4 short paragraphs.]

[The practical takeaway / the gap to close.]
[https://bitcoinsovereign.academy]
```

## 9. family
Plain language for families holding Bitcoin. No jargon.
```
[A reassuring-but-honest reframe of a family worry.]

[The pieces a real plan needs — in everyday words.]
```

## 10. spanish
Native LATAM Spanish. Not a literal translation. Explain terms naturally
(no forced "auto-custodia" / "reserva de valor" calques).
```
[Una creencia común, en una línea.]

[La verdad más útil.]

[1–3 líneas de contraste o ejemplo concreto.]
```

---

### CTA discipline
- Most posts: `cta_type: none`.
- BSA lesson link: 3–4 / week.
- Zap-soft: ≤ 2 / week (set `zap_ok: true`).
- Meeting CTA: ≤ 1 / week (set `meeting_ok: true`, link `https://meetings.hubspot.com/dalia-platt`).
- Never the same closing line more than twice across a batch.
