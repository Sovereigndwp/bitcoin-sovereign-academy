# Template — Main post (kind 1)

**Length:** 1–6 short lines. One idea. No headers.
**Hashtags:** 0–2 max. Often zero.
**Links:** 0–1. Never link in the first line.

## Pattern A — Mistake correction

```
<one short sentence stating the wrong belief>.

<one short sentence stating the more useful belief>.

<3–5 short lines showing the contrast or the cost>.

<optional: one link OR one question, not both>.
```

## Pattern B — First principle

```
<a noun is not what most people think it is>.

<what it actually is, in plain language>.

<one concrete consequence that flips after this realization>.
```

## Pattern C — Checklist

```
<topic in one line>.

— <item 1>
— <item 2>
— <item 3>
— <item 4>

<one short closing line that makes the point>.
```

## Pattern D — Open question

```
<one strong question that has a real answer, not rhetoric>.

<one line of context for why the question matters>.

<no answer in this post — let people reply>.
```

## Filter (run before publishing)

- Useful without clicking?
- One idea, not three?
- No slogans?
- Strong first line?
- A reader who knows nothing about BSA still learns something?
