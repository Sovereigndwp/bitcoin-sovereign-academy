Sub-project #2.7 · Phase 1 prototype

Icon system — review before about.html lands

Custom icon prototypes drawn from the locked diamond/stroke logo geometry. Tier 1 (brand-anchor) only — these would replace the 5 emojis stripped from about.html: the page title slot plus 4 Core Values cards. Two variants per concept where the geometry has alternatives — pick a winner per concept, or mix.

How to review

Each card shows two design variants of the same concept. A · recommend marks my preferred direction. All icons render at the actual about.html usage size — 48px for value cards, 32px for the page title slot. They use currentColor so they inherit text color (shown here in brand orange #C8922A; on dark surfaces they'd inherit white or the dark-mode accent). Once you pick, I implement on about.html as the Phase 1 commit.

1 · Page title slot — "About" (round 2)

Round 1 (diamond+stroke and solid diamond) was rejected. Trying again — and seriously considering "no icon at all" as the right answer. Locked spec §7.4 says icons "used only where they reduce reading load." A page h1 is already short and scannable, and the eyebrow above ("Don't Trust, Verify.") already carries brand presence. Adding an icon to the h1 may be working against the page, not for it. A · no icon · recommend

Variant A · no icon · pure h1 recommend
Don't Trust, Verify.

About Bitcoin Sovereign Academy

No icon. Eyebrow does the brand work above; h1 carries the title. Cleanest, most editorial — and per spec §7.4, the right answer when an icon doesn't reduce reading load.

Variant B · Dalia stroke through ghost diamond

About Bitcoin Sovereign Academy

The Dalia mark — bold stroke through a ghost-weight diamond. Per CLAUDE.md, this is the locked artifact for Dalia-only contexts. Reads as "Dalia's signature on this." Personal, not branded BSA-vs-FSA.

Variant C · diamond with center dot (origin / kernel)

About Bitcoin Sovereign Academy

Diamond outline + a single point at the center. Reads as "the source / origin / what BSA is at its core." Less metaphorically loaded than the BSA logo — feels more like an "about" page.

Variant D · layered diamonds (depth / inside)

About Bitcoin Sovereign Academy

Outer diamond + smaller inner diamond at 70% scale, ghosted. Reads as "looking inside / behind the surface" — the meta-meaning of an About page. Geometric, brand-aligned, low-key.

If you don't like any of these either, the answer is probably A (no icon). The Core Values cards (sections 2–5) genuinely benefit from icons — 4 cards = visual scannability. A single h1 with an eyebrow above doesn't need decoration.
