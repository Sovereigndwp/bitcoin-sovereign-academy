Map your UTXOs with Sparrow

A 5-minute walk-through. The prerequisite for any privacy decision you make about your Bitcoin.

This guide is being written

What this guide will cover (planned outline)

  1. Install Sparrow Wallet (download + verify signature).
  2. Add your existing wallet to Sparrow as watch-only via xpub. No private keys touch Sparrow.
  3. Open the UTXOs view; identify your 5 largest UTXOs.
  4. Label each one with where it came from (exchange withdrawal, salary, gift, etc.).
  5. Read the resulting map: which counterparties can already see which coins.

Outcome: you'll have a UTXO map — the input that any further privacy work depends on.
