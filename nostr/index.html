That matches what BSA already is: free Bitcoin education that doesn't depend on a centralized gatekeeper.

How to start on Nostr in 5 minutes

  1. Install a client. iOS: Damus. Android: Amethyst. Web: Primal or Snort.
  2. Let the app generate your keys. Save the secret key (nsec…) in a password manager. It's the only way back in. No password reset.
  3. Search bitcoinsovereign.academy. Follow the account with the verified badge.
  4. Optional: connect a Lightning wallet inside the client so you can send and receive zaps. Coinos and Alby are good starting points.
