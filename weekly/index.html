    Security Fix — API Data Sanitized Before DOM Injection

    Audited all new JS files for XSS risk. Found that bitcoin-context.js interpolated raw mempool.space API values directly into innerHTML. Fixed by adding safeInt() and safeNum() helpers that coerce all API values to numbers before any DOM injection — preventing string injection from a compromised API response.
