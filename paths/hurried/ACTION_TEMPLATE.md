# The Hurried Path - Action-Packed Module Template

## Module Structure (10 minutes each)

### WHY (30 seconds - 1 minute)
- Single compelling reason
- One horror story OR one success story
- No theory, just stakes

### HOW (7-8 minutes)
- Numbered steps with screenshots
- Specific tools/apps named
- Checkboxes for completion
- Video walkthrough embedded
- Troubleshooting section

### DO IT NOW (1 minute)
- Completion checklist
- Verification step
- Next module preview

### NO FLUFF RULES:
❌ No history lessons
❌ No "understanding how it works"
❌ No philosophy
❌ No comparison tables (unless 3 rows max)
❌ No quizzes or reflection questions
✅ ONLY: Do this → Verify → Next

---

## Module 1: Buy Bitcoin (10 min)
**Goal:** $20-100 Bitcoin in exchange account

**WHY:** You can't secure what you don't own. Start here.

**HOW:**
1. Choose ONE exchange (pre-vetted)
   - Strike (fastest, US)
   - River (best for learning)
   - Coinbase (most familiar)
2. Create account (2 min)
3. Verify identity (3 min)
4. Buy $20-100 (2 min)
5. Screenshot your balance

**DONE:** ✅ Bitcoin purchased

---

## Module 2: Move to Self-Custody (10 min)
**Goal:** Bitcoin in YOUR wallet, off exchange

**WHY:** Mt. Gox, FTX, Celsius. Get your Bitcoin off exchanges.

**HOW:**
1. Download BlueWallet or Muun
2. Create new wallet
3. Copy receive address
4. Go to exchange → Withdraw
5. Paste address → Send
6. Wait for confirmation (track on mempool.space)

**DONE:** ✅ Self-custody confirmed

---

## Module 3: Secure Seed Phrase (10 min)
**Goal:** Backup written, tested, stored

**WHY:** Lose seed = lose Bitcoin. Forever. No customer service.

**HOW:**
1. Write 12-24 words on paper (pen, NOT digital)
2. Number them 1-12
3. Store in fireproof location
4. Create second copy → different location
5. Test: Restore wallet on second device
6. Verify balance matches

**DONE:** ✅ Seed backed up and tested

---

## Module 4: Test Recovery (10 min)
**Goal:** Prove you can recover if phone dies

**WHY:** 99% think they backed up correctly. Only testing proves it.

**HOW:**
1. Delete wallet app (or use second device)
2. Reinstall app
3. Select "Restore from seed"
4. Enter 12 words in exact order
5. Verify balance appears
6. Send 0.0001 BTC test transaction

**DONE:** ✅ Recovery verified

---

## Module 5: Send Transactions (10 min)
**Goal:** Move Bitcoin without losing it

**WHY:** One wrong address = permanent loss

**HOW:**
1. Pre-send checklist:
   - ✅ Triple-check address
   - ✅ Start with $5-10 test
   - ✅ Use bc1 addresses (lowest fees)
   - ✅ Check mempool.space for fee rates
2. Send test transaction
3. Monitor on block explorer
4. Wait for 1 confirmation
5. Send full amount (if test succeeded)

**DONE:** ✅ Transaction completed

---

## Module 6: Upgrade to Multisig (10 min)
**Goal:** Understand when/how to upgrade security

**WHY:** Single point of failure. Multisig = multiple keys required.

**HOW:**
1. When to upgrade: >$10,000 in Bitcoin
2. What is multisig: 2-of-3 keys required
3. Options:
   - Casa ($10/mo, managed)
   - Unchained (collaborative)
   - Sparrow (DIY, advanced)
4. Bookmark setup guides
5. Graduate!

**DONE:** ✅ Path completed
