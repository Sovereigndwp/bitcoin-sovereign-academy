# Purpose

Use this skill to improve existing educational content without losing the original teaching goal.

# Inputs

Target file or lesson name: $0

Additional user context:
$ARGUMENTS

# What to do

1. Review the existing content first.
2. Identify:
   - repeated ideas
   - weak transitions
   - too much abstraction
   - unnecessary jargon
   - sections that do not move learning forward

3. Preserve the main teaching goal.

4. Improve the content using this order:
   - sharpen the headline if needed
   - improve section sequence
   - remove redundancy
   - simplify explanations
   - add stronger examples where needed
   - make the next step clearer

5. Do not make the content sound generic, robotic, or overly polished.

6. Keep the voice direct, human, and useful.

# Output requirements

Return:
- what is weak
- what should change
- improved structure
- revised content or edits

# Quality bar

The final version should:
- teach better
- read faster
- feel more human
- remove repetition
- preserve intent
