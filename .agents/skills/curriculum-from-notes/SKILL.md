# Purpose

Use this skill to transform rough educational notes into an organized learning experience.

# Inputs

Topic or source notes:
$ARGUMENTS

# What to do

1. Extract the distinct ideas.
2. Group related ideas.
3. Put them in the strongest teaching order.
4. Identify:
   - what belongs together
   - what is repetitive
   - what should be a module
   - what should be an activity
   - what should be a takeaway

5. Build a curriculum structure with:
   - module titles
   - objective
   - core idea
   - suggested activity
   - practical takeaway

6. Remove redundancy and keep momentum between modules.

# Output requirements

Return:
- curriculum map
- recommended sequence
- module summaries
- notes on gaps or overlap

# Quality bar

The result should feel teachable, logical, and easy to expand later.
