# Purpose

Use this skill to make a page easier to discover and easier to understand.

# Inputs

Target page: $0
Primary keyword or topic: $1

# What to do

1. Review the page for:
   - page intent
   - headline clarity
   - heading structure
   - repeated or weak sections
   - internal linking opportunities
   - skimmability
   - mismatch between audience intent and copy

2. Improve human readability first.
3. Strengthen search clarity second.
4. Do not stuff keywords.
5. Keep the page natural and useful.

6. Suggest:
   - stronger title
   - improved headings
   - meta description idea
   - internal links to add
   - sections to tighten or expand

# Output requirements

Return:
- SEO and structure issues
- revised heading outline
- suggested title and meta description
- copy improvements if requested

# Quality bar

The page should be better for humans and clearer for search.
