# Purpose

Use this skill to create pages that speak clearly to institutional or professional users.

# Inputs

Page topic: $0
Audience: $1
Primary CTA: $2

# What to do

1. Identify:
   - who the page is for
   - what problem they have
   - why they should care now
   - what action they should take next

2. Build the page in a professional flow:
   - strong headline
   - credibility or trust setup
   - problem framing
   - solution overview
   - key benefits
   - tradeoffs or honest nuance if relevant
   - clear CTA

3. Keep the tone:
   - professional
   - grounded
   - direct
   - non hype

4. Avoid playful educational tone unless explicitly requested.

5. Reuse existing layout and CTA patterns when possible.

# Output requirements

Return:
- page structure
- recommended sections
- suggested copy blocks
- files to create or edit
- CTA placement notes

# Quality bar

The page should feel credible, clear, and easy to act on.
